《外语学习中的真实方法和误区》读书笔记

发表于 2016-01-24   |   分类于 读书笔记   |  

来源

来自于天涯上的一篇帖子，详细介绍了英语学习中的误区，以及正确（或者说更高效）的方法。读了这本书之后感觉到自己在英语学习的道路上走了很多弯路。总结一下，希望在未来的英语学习道路上能够找到正确的方向。

tips

发音不标准

不应该去听CNN，BBC等所谓纯正英语，而是需要听多种口音，尝试分辨其中的区别。说的准，关键在于听得准。

用语言思考

会用任何语言交流的前提是必须能用这种语言思考

小孩学的更快

  • 一般到英文环境中从不流利到流利是年龄除6。
  • 年龄越大学习英语越困难，并不是因为丧失了天赋，是因为随着年龄的增长，会越来越习惯于使用记忆去学习英语。

学习的习惯

  • 儿童学习英语的过程，猜测，图像联系，条件反射，英语思维
  • 语言不是记忆，所以通过记忆学习语言效率很低
  • 学习时脱离语法翻译，而是交流应用

英语学习流派

  • 语法-翻译，通过学习语法，进行翻译来学习英语
  • 第二语言习得理论，在学习过程中，学习者通常意识不到 自己在习得语言，而只在自然交际中不知不觉地学会了第二语言。

五大假设

第二语言习得理论，五大假设

  • 习得——学得差异假设(The Acquisition-Learning Hy-pothesis)
  • 监检假设(The Monitor Hypothesis)
  • 输入假设(The Input Hypothesis)
  • 情感过滤假设(The Affective Filter Hypothesis)
  • 自然顺序假设(The Natural Order Hypothesis)

学习英语的四个时期

早期，1-2month

  • 只听不说，看图识音，建立条件反射，1000词汇量+短语量，推荐软件，罗塞达石碑
  • TPR，全身反应法，词汇1000个左右。TPR-O(Object)，TPR-P（Pictures），TPR-S（Stories）

提高期（听）

1． 要学会听懂单句子，建立整句的英文思维；
2． 达到能听懂简单的，可能是慢语速的整段英文；
3． 在提高期后期开始尝试简单“说”英语，实现简单沟通；
4． 开始引入阅读材料（不是学习如何阅读）；

  • 可以了解一点英文语法，但只关注于与中文的不同点

  • 材料准备很关键，提供可理解性输入i+1
    （1）序列法，1输入i+1，一系列的整句+动作
    （2）生成对话，听人家对话

  • 北京人在纽约，纽约人在北京，学习语言的差异
    中国人很乐于与外国人交流（care-taker speaker，正负刺激 ），耐心调整语言难度，容易形成i+1难度，而外国人通常不会

  • 单词和拼写
    对于常用的5000单词，需要建立英文思维，5000-15000出于实用主义可以背，这10000-15000单词可以通过词根记忆进行背诵

  • 使用词根背单词
    1， 英语单词中，80%以上由词根构成，而且刚好是复杂词（2000 个基础词以上）。越简单越不用词根，比如 this, that, big ,bag.
    2， 同一词根使用的单词在几个到几十个之间，会一个就会了一串；
    3， 常用英文词根有几百个，只需要250个左右就可以了。而且这250个中，有大约100个大家已经认识了，可能没意识到；
    4， 每天不要贪多，用30分钟就记100个，再用10分钟稍微看一下昨天的。三个月一到，10000个拿下，才花了60小时。

  • 关于阅读
    阅读的定义是按说出的语言的声音规律进行图象材料的解码，而该语言的声音在阅读开始前就应该已经掌握了。

  • 学习英语的关键
    学习外语的成功要素主要有三个，一是方法，二是愿望，三是自律

增长期

  • 听力过程的可能出现的问题
    1， 内部声音库和声音辨别的问题：(acoustic reason)
    2， 解析过程 (Parsing)
    3， 文化知识库 (cultural knowledge bank)
    4， 头脑中的“母语翻译”

  • 听力策略：
    1，“从上至下”的听力处理 (Top-down Processing)，主题
    2，“自下而上”的处理过程 Bottom-up Processing ，细节

  • 关键是听力材料的选取，i+1
    看英文电影，不能开字幕，可以先看剧本，可以多看几遍，
    不能写，建立声音-含义的联系，联系使用用语言思维
    使用目标语言思维VS使用母语思维：
    使用母语思考降低阅读速度，能够提升理解深度。阅读和演讲使用目标语言思维，写作时使用母语思维。总是要尽力避免翻译过程

  • 通用学习原则
    1， 需要建立外语思维： Able to think in the target language
    2， 语言的掌握不是通过翻译和记忆，而是与思维，概念或图像建立直接联系：Through association, not translation or memorization
    3， 语法的掌握是下意识的过程，不应该通过有意识地学习： An implicit subconscious learning process rather than explicit conscious learning or active instruction.
    4， 要注重理解和含义而非结构：Meaning-focus instead of form-focus.
    5， 语言环境和掌握语言的关系：Linguistic environment and language acquisition
    6， 关键是提供可理解输入条件 Comprehensive input i+1

  • Narrow Input:
    把输入材料限制在比较窄的范围更有用，可以用高熟悉度集中提高

  • 语法学习
    语法其实也是如此，应该用 20%的精力介绍一些最基本的几个时态变化就足以， 没必要再复杂了，通过语言本身掌握就得了。

  • 关于背诵
    Rote Learning， 简称 RL，机械师重复取得记忆。语言不是存储在记忆区，而是反射区。
    使用（1）联系式记忆，故事情节（2）图像和声音记忆

  • 结束
    第三阶段完成就可以达到能够进行正常的交流和沟通，开始在英文环境工作和学习没有太大困难， 看英文电影能理解大部分。尽管对复杂的英文理解有困难或还有些听不懂的部分，说的英文大多是大白话，
    但如果听说的反应迅速，已经可以定义为流利的英文了。

高级期

  • 高级期已经可以应付基本的生活和工作需要。达到母语水平是不太可能的，但是仍然有提高的空间。
  • 首先是心理上的障碍，对于更高难度的材料往往是拒绝的。要学着接受i+1的难度。
  • 学习的点不再是语言本身的结构，而是关于各种文化背景知识的了解。
  • 深度阅读是非常有效的手段，可以找英文教材来看
